# 使用说明

## 1. 客户怎么使用

客户访问前台：

```text
/zh
/ja
/en
```

客户流程：

```text
注册账号
↓
登录
↓
进入云手机控制台
↓
查看自己的云手机
↓
点击打开云手机
↓
系统通过后端申请临时 Token
↓
网页打开云手机画面
```

客户页面：

```text
/zh/register
/zh/login
/zh/app
/zh/app/phones
/zh/app/orders
/zh/app/support
```

## 2. 管理员怎么使用

管理员入口：

```text
/zh/admin
```

管理员账号来自 Coolify 环境变量：

```env
ADMIN_EMAIL
ADMIN_PASSWORD
```

管理员页面：

```text
/zh/admin/users
/zh/admin/orders
/zh/admin/plans
/zh/admin/phones
/zh/admin/assignments
/zh/admin/settings/vmos
/zh/admin/support
/zh/admin/logs
```

## 3. 导入 VMOS 手机

进入：

```text
/zh/admin/phones
```

填写：

```text
内部名称：JP-001
VMOS padCode：从 VMOSCloud 获取
地区：Japan
VMOS 到期时间：上游服务到期时间
备注：内部备注
```

## 4. 分配手机

进入：

```text
/zh/admin/assignments
```

选择：

```text
客户
订单
空闲手机
客户到期时间
```

保存后，客户在自己的云手机控制台可以看到这台设备。

## 5. 配置 VMOSCloud

进入：

```text
/zh/admin/settings/vmos
```

填写：

```text
API Base URL: https://api.vmoscloud.com
Token Path: /vcpcloud/api/padApi/stsTokenByPadCode
网页连接 baseUrl: https://api.vmoscloud.com
AccessKey / AK
SecretKey / SK
```

密钥会加密保存，不会显示给客户。

## 6. 客服工单

客户在：

```text
/zh/app/support
```

提交问题。

管理员在：

```text
/zh/admin/support
```

回复或关闭工单。

## 7. 套餐管理

管理员在：

```text
/zh/admin/plans
```

可以：

```text
新增套餐
编辑套餐
上架 / 下架套餐
```

## 8. 重要提醒

正式运营必须满足：

```text
数据库正常
管理员账号已创建
VMOSCloud AK/SK 已配置
VMOS padCode 已导入
客户订单已付款
手机已分配给客户
```

如果注册失败，先打开：

```text
/api/health
```

看 `database` 是否是 `ok`。
